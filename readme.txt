-gettweet
タイムラインからツイートを取得する。
API制限を考えて使うべし

-raw_toinput
ツイートを単語ごとに分割する。後の処理の準備

-wordcut
分割された文章を3単語づつで区切って保存する。
マルコフ連鎖で文章を作る準備

-wordconnect
マルコフ連鎖で文章を作る。

-tweet
完成した文章をツイートする。

-open3
上記の5つのスクリプトを連続で実行する。
この順番で実行しないと上手くいかないよ

-followback
フォロワーリストを取得してフォローする。
